# System Threat Forecaster (ML Project)

## Overview
This project predicts system threats using machine learning techniques based on system activity data.

## Problem Statement
To classify and forecast potential system threats using historical data.

## Tech Stack
- Python
- pandas, NumPy
- scikit-learn

## Approach
- Data preprocessing and cleaning
- Feature engineering
- Model training (classification)
- Model evaluation

## Result
Achieved a model score of 0.63 on the validation dataset.

## Learnings
- Improved understanding of classification models
- Hands-on experience with real-world dataset
- Model evaluation techniques
